### **Method 1: Using Android Studio**
1. Open the project in Android Studio
2. Make sure you have an Android device connected with USB debugging enabled
3. Click the "Run" button (green play icon)
4. The app will build and install automatically

### **Method 2: Using Command Line** (if you have Java/Gradle set up)

## 🚨 Troubleshooting

### **AI Not Responding?**
- Check that `local.properties` contains your OpenAI key
- Verify internet connection
- Look for "API key not configured" message

### **No Data Showing?**
- Sample data seeds automatically on first run
- If dashboard is empty, restart the app once

### **No Notifications?**
- Grant notification permissions when prompted
- Check Android notification settings for the app
